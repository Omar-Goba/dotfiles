## в—Ҷ launch sequence

<table>
<tr><td>

```bash
# 1 в”Җ clone into place
git clone https://github.com/Omar-Goba/dotfiles ~/dotfiles

# 2 в”Җ wire it up
ln -sf ~/dotfiles/config/nvim ~/.config/nvim
ln -sf ~/dotfiles/config/tmux ~/.config/tmux
echo 'source ~/dotfiles/shell/core.zsh' >> ~/.zshrc

# 3 в”Җ take off
exec zsh
```

</td></tr>
</table>

> <sub>keep machine-specific bits in `shell/local.zsh` вҖ” already ignored, stays off the public record.</sub>

## в—Ҷ public-push safety

This repo is intended for public hosting. Secrets, keys, tokens and machine-private values never get committed.

```gitignore
.env
.env.*
*.pem
*.key
secrets*
shell/local.zsh
shell/secrets.env
```
